A desktop client for Hacker News

Comfortably read Hacker News without having to open an endless amount of tabs anymore. You can choose articles to view from a sidebar and can optionally read articles and their comments in a split view next to each other

View on GitHub Downloads

  • Switch between stories using the sidebar

    You can just choose a story on the sidebar and the corresponding website will load inside the app. No more needing to open an endless amount of tabs for HN and you can switch to the comments using a keyboard shortcut.

  • View website and comments next to each other

    Optionally you can use a split screen to view website and the comments next to each other. It's an awesome way to read HN if your screen is large enough.

  • Easily read the comments

    By pressing a keyboard shortcut you can switch to viewing the comments right in the app. They were prefetched while you read the article. There are also keyboard shortcuts to navigate through the comments and you can fold comments.

  • Switch between different HN lists

    You can go through different story lists, just like on HN. For example you can go through all the latest "Show HN" or "Ask HN" posts. Unlike other apps this app gets the details right, like directly showing the comments for "Ask HN" posts, instead of rendering an empty website.

Feature overview

There's a lot of small things the app gets right

  • Auto loading new articles

    New articles are automatically loaded if you scrolled down far enough on the "Top Stories" list

  • Keyboard shortcuts

    You can easily navigate the app using Vim like keyboard shortcuts

  • Polls

    Things that other HN apps usually forget are included, e.g. viewing HN polls right inside the app

  • Readability included

    It's terrible to read articles on some website but you can use Readability inside the app to fix that

  • Awesome comment navigation

    You can navigate the comments using keyboard shortcuts and fold them which makes a great comment reading experience

  • Website / comment split screen

    You can view website and comments next to each other or cycle between the different display modes using a keyboard shortcut

  • Read stories are remembered

    Read stories are marked as such, just like in the browser

  • PDF support

    Reading PDFs inside the app is no problem, it's automatically detected when to display a PDF reader

  • It's Open Source!

    It's open source and on GitHub, you can contribute to it or just change it however you like :)
